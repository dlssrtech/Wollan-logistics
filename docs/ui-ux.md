# UI/UX Screens and Wireframes

## Design System

- Framework: Next.js, React, TypeScript, Tailwind CSS, Redux Toolkit.
- Mobile: Flutter with shared design tokens for color, spacing, typography, and component states.
- Accessibility: WCAG 2.2 AA contrast, keyboard navigation, semantic landmarks, screen-reader labels, focus rings.
- Languages: English, Hindi, and extensible locale files for city-specific language rollout.
